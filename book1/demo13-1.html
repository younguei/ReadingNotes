unicode可以利用font-size调整字体大小

            @font-face {
                font-family: 'example';
                src: url(example.ttf);
                font-style: normal;
                font-weight: normal;
                unicode-range: U+0025-00FF;
                font-variant: small-caps;
                font-stretch: expanded;
                font-feature-settings："liga1" on;
            }
            .font {
                font-family: example;
            }

@font-face是css3的新属性，表示嵌入字体，加载服务器端的字体文件，让浏览器端可以显示用户电脑没有安装的字体
@font-face{font-family:字体名称;src:url("字体文件路径");} 这里font-family看成一个字体变量，名称随意
