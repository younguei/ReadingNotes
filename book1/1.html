第三章:CSS基础知识

关于em:单位的问题 em是相对于当前元素字体大小而言的。

            .parent{
                font-size:10px;
            }
            .child{
                font-size:1.5em;
                height: 10em;
                width: 5em;
            }

这里子元素的宽高为150px 75px em单位的大小是根据当前元素的字体大小来确定的，此时font-size为15px;所以1em为15px,不是10px

所以：当子元素字体大小改变后，元素em需要重新计算

第四章:CSS规范

1. 关于CSS样式书写的先后顺序 影响文档流样式 > 元素盒子模型样式 > 文本样式 > 装饰样式 > 其它样式

            .demo{
                /*影响文档流样式*/
                display:block;
                positon: absolute;
                left: 200px;
                top: 200px;
                /*盒子模型*/
                width:400px;
                height:400px;
                /*文本样式*/
                vertical-align: center;
                font-size:20px;
                /*装饰样式*/
                background:red;
                color:#ccc;
                /*其它*/
                cursor: pointer;
            }

2. 关于reset.css样式 重置css默认样式 *{padding:0;margin:0}效果差，实际项目中一般使用reset.css来重置样式

第九章:浮动布局

1. 正常文档流 脱离文档流(利用浮动float或定位postion来改变默认位置)
2. 浮动的影响

  • 元素自身变为block元素，可以设置margin width height
  • 当父元素高度未定义或小于子元素高度，父元素高度会塌陷
  • 对兄弟元素的影响: 浮动于正常流之上
  • 当父元素也是浮动元素，子元素也是浮动元素，那么父元素会自动包含该子元素
3. 清除浮动
  • clear:both; clear属性用于浮动元素后面的元素 实例链接
  • overflow:hidden; 在父元素内添加该属性，缺点是会隐藏超出父元素的内容 实例链接
  • ::after伪元素 推荐使用这个方法 用来创建一个伪元素，作为已选中元素的最后一个子元素。通常配合content属性来为该元素添加装饰内容。这个元素默认为行内元素 伪类元素详解
